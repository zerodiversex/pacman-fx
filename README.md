C’est un projet de développé un jeu vidéo simple, il s'appelle Pacman.

## Technologie
Projet est créé avec:
#### Framework:
    * JavaFX
    
#### Langages:
    * Java
    
#### Algorithm:
    * CreatMath
    
## Installer
Pour exécuter ce projet, installez-le localement à l'aide de npm :
```
$ Click the Run application icon
$ Run .main().
```
